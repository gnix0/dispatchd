## [v0.1.0] - 2026-04-10

### Added

- distributed orchestration across `control-plane`, `scheduler`, and `worker-gateway`
- shared Postgres and Redis state for jobs, executions, worker state, and scheduling coordination
- Kubernetes, GitOps, and Argo CD deployment assets
- DevSecOps workflows for code scanning, dependency scanning, container scanning, secret scanning, and DAST
- observability stack with Prometheus, Grafana, Jaeger, and performance evidence assets
- contributor, security, and release governance documentation
